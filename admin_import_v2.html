📦 Admin Import v2

Загрузка Excel/CSV напрямую в SQL через безопасную RPC-функцию Supabase. Повторяет логику листа «Загрузка»: валидация, агрегация qty, upsert заказов и позиций.
🔒 Сессия не проверена

Управление импортом

1
Подключение
Используется publishable key. Для RLS лучше войти как администратор.
2
Файл заказов
Поддерживаются .xlsx и .csv. Первый лист считается источником данных.
Выбрать файл Файл не выбран
3
Правила проверки
Строгая проверка SKU по sku_catalog
Строгая проверка ПВЗ/филиала
Если включено — импорт остановится при неизвестном SKU или ПВЗ.
